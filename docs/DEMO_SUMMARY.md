# Convert Pro 3 - 시연 요약

## 🎯 프로그램 소개 (30초)

**Convert Pro 3**는 로거 데이터를 자동으로 변환하고 관리하는 시스템입니다.

- 여러 업체/현장의 로거 파일을 체계적으로 관리
- 자동 변환 및 스케줄링
- 직관적인 UI로 쉬운 사용

---

## 🖥️ 화면 구성 (1분)

### 메인 화면 5개 영역

1. **상단 헤더**
   - 업체 선택 콤보박스
   - 새로고침 버튼

2. **중앙 트리뷰** (핵심)
   - 업체 → 현장 → 폴더 → 파일 (4단계 구조)
   - 비고, 배터리 정보 표시
   - ⚠️ 아이콘: 경로 유실 항목

3. **상태 영역**
   - 진행률 바
   - 상태 메시지

4. **기능 버튼** (우측)
   - ▶ 변환 실행
   - 업체 관리
   - 현장 추가
   - 로거파일 등록
   - 미등록 파일

5. **로그창** (하단)
   - 실시간 작업 로그

---

## 🔧 핵심 기능 데모 (5분)

### 1️⃣ 업체 관리 (30초)
- **버튼**: "업체 관리"
- **기능**: 업체 추가/삭제/순서 변경
- **데모**: 업체 추가 → 목록 확인

### 2️⃣ 현장 추가 (30초)
- **버튼**: "현장 추가"
- **기능**: 업체 선택 → 현장 이름 입력
- **데모**: 새 현장 추가 → 트리뷰 확인

### 3️⃣ 로거파일 등록 (1분)
- **버튼**: "로거파일 등록"
- **기능**: 폴더 선택 → 미등록 목록에 추가
- **중요**: 변환 대상 아님 (미등록 상태)
- **데모**: 폴더 선택 → 미등록 목록 확인

### 4️⃣ 미등록 파일 관리 (1분)
- **버튼**: "미등록 파일"
- **기능**: 
  - 미등록 파일 목록 확인
  - 파일 선택 (체크박스)
  - 현장 선택하여 등록
- **데모**: 파일 선택 → 현장 등록 → 트리뷰에 표시

### 5️⃣ 파일 변환 (1분)
- **버튼**: "▶ 변환 실행"
- **기능**: 
  - 등록된 모든 파일 변환
  - 멀티스레딩 처리 (빠른 속도)
  - 진행률 표시
- **데모**: 변환 실행 → 진행률 확인 → 완료

### 6️⃣ 채널 설정 (1분)
- **접근**: 파일 더블클릭
- **기능**: CH0~CH7 채널별 센서 설정
- **데모**: 파일 더블클릭 → 채널 설정 변경 → 저장

## ❓ 예상 질문 대비

**Q: 미등록 파일과 정식 등록의 차이는?**
A: 로거 등록 시 미등록 목록에만 추가 (변환 안 됨). 현장 등록 시 정식 등록되어 변환 대상이 됩니다.

**Q: 변환 속도는?**
A: 멀티스레딩으로 4개 파일 동시 처리. 증분 변환으로 이전 데이터는 스킵합니다.

**Q: Ghost 상태란?**
A: 경로가 유실된 항목. ⚠️ 아이콘으로 표시되며, 경로 재지정 필요합니다.

**Q: 배터리 정보는?**
A: 각 파일의 배터리 상태를 자동으로 읽어 표시합니다.
