# Convert Pro 3 - 시연 가이드

## 📋 프로그램 개요

**Convert Pro 3**는 로거 데이터를 변환하고 관리하는 시스템입니다.

### 주요 목적
- 로거 CSV 파일을 표준 형식으로 변환
- 여러 업체/현장의 데이터를 체계적으로 관리
- 자동화된 변환 및 스케줄링 기능 제공

---

## 🖥️ 메인 화면 구성

### 1. 상단 헤더 영역
- **프로그램 제목**: "Convert Pro 3 - 데이터 변환 관리 시스템"
- **업체 선택 콤보박스**: 전체 업체 또는 특정 업체 선택
- **새로고침 버튼**: 트리뷰 및 업체 목록 갱신

### 2. 중앙 트리뷰 영역
**4단계 계층 구조**:
```
업체 (Company)
  └── 현장 (Site)
      └── 폴더 (Folder)
          └── 파일 (CSV 파일)
```

**표시 정보**:
- **파일/폴더/현장**: 트리 구조로 표시
- **비고**: 각 항목의 메모/설명
- **배터리 (%)**: 파일별 배터리 상태 표시

**특수 표시**:
- ⚠️ 아이콘: 경로가 유실된 항목 (Ghost 상태)

### 3. 상태 및 진행률 영역
- **상태 메시지**: 현재 작업 상태 표시
- **진행률 바**: 변환 작업 진행률 시각화

### 4. 기능 버튼 영역 (우측)
1. **▶ 변환 실행** (파란색) - 전체 파일 변환 시작
2. **업체 관리** (초록색) - 업체 추가/삭제/순서 변경
3. **현장 추가** (회색) - 새 현장 추가
4. **로거파일 등록** (회색) - 로거 파일을 미등록 목록에 추가
5. **미등록 파일** (주황색) - 미등록 파일 관리

### 5. 로그창 (하단)
- 실시간 작업 로그 표시
- 시간 정보 포함

### 6. 상태바 (최하단)
- 현재 상태 요약 정보

---

## 🔧 주요 기능

### 1. 업체 관리
**접근**: 상단 버튼 → "업체 관리"

**기능**:
- ✅ 업체 목록 조회
- ✅ 업체 추가
- ✅ 업체 삭제 (하위 현장/폴더/파일 모두 삭제)
- ✅ 업체 순서 변경 (위로/아래로 이동)

**UI 특징**:
- 리스트박스로 업체 목록 표시
- 직관적인 버튼 배치
- 메인 창 중앙에 팝업 표시

---

### 2. 현장 추가
**접근**: 상단 버튼 → "현장 추가"

**기능**:
- 업체 선택 (팝업)
- 현장 이름 입력
- 선택한 업체에 현장 추가

**UI 특징**:
- 커스텀 팝업 (450x200)
- 헤더 스타일 통일
- Enter 키로 확인 가능

---

### 3. 로거파일 등록
**접근**: 상단 버튼 → "로거파일 등록"

**기능**:
- 폴더 선택 다이얼로그
- 선택한 폴더의 CSV 파일을 미등록 목록에 추가
- **중요**: 변환 대상이 아님 (미등록 상태로만 저장)

**사용 흐름**:
1. "로거파일 등록" 버튼 클릭
2. 로거 파일이 있는 폴더 선택
3. 자동으로 미등록 파일 목록에 추가됨
4. 미등록 파일 관리 화면으로 이동

---

### 4. 미등록 파일 관리
**접근**: 상단 버튼 → "미등록 파일"

**기능**:
- 미등록 파일 목록 표시
- 파일 정보: 파일명, 폴더, 수정일시, 크기
- 다중 선택 (체크박스)
- 현장 선택하여 등록

**UI 구성**:
- TreeView로 파일 목록 표시
- 체크박스로 선택/해제
- 전체 선택/해제/토글 버튼
- "현장 선택하여 등록" 버튼

**사용 흐름**:
1. 미등록 파일 목록 확인
2. 등록할 파일 선택 (체크박스)
3. "현장 선택하여 등록" 클릭
4. 업체 및 현장 선택
5. 등록 완료 → 트리뷰에 표시됨

**특징**:
- 로거 등록 시: 미등록 목록에만 추가 (변환 대상 아님)
- 현장 등록 시: 정식 등록되어 변환 대상이 됨

---

### 5. 파일 변환
**접근**: 상단 버튼 → "▶ 변환 실행"

**기능**:
- 등록된 모든 파일을 변환
- 멀티스레딩 처리 (4개 워커)
- 증분 변환 (이전 변환 이후 데이터만 추가)
- 진행률 표시

**변환 과정**:
1. 원본 CSV 파일 읽기
2. 데이터 가공 (센서 처리, 배터리 이동 등)
3. 표준 형식으로 변환
4. 변환본 저장 (C:\data\Convertfile)

**UI 피드백**:
- 진행률 바 업데이트
- 상태 메시지 변경
- 로그창에 상세 정보 표시
- 변환 중 버튼 비활성화

---

### 6. 채널 설정
**접근**: 트리뷰에서 파일 더블클릭

**기능**:
- CH0~CH7 채널별 설정
- 센서 모드 선택 (PASS, OFFSET, SCALE 등)
- 파라미터 설정 (base, scale, initial 등)
- 전역 옵션 (fill_interval, gen_interval)

**UI 구성**:
- 8개 채널 탭
- 각 채널별 설정 필드
- 툴팁으로 도움말 제공
- 저장/취소 버튼

---

### 7. 우클릭 메뉴
**접근**: 트리뷰 항목 우클릭

**기능** (항목 타입별):
- **현장**: 비고 편집, 삭제
- **폴더**: 비고 편집, 삭제
- **파일**: 비고 편집, 채널 설정, 삭제

---

## 📊 데이터 구조

### 계층 구조
```
업체 (Company)
  └── 현장 (Site)
      └── 폴더 (Folder)
          └── 파일 (CSV)
              └── 채널 설정 (CH0~CH7)
```

### 설정 파일
- **config.json**: 메인 설정 (업체/현장/폴더/파일 구조)
- **unregistered_files.json**: 미등록 파일 목록

## 🎯 주요 사용 시나리오

### 시나리오 1: 새 로거 파일 등록
1. "로거파일 등록" 클릭
2. 폴더 선택
3. 미등록 목록에 자동 추가 확인
4. "미등록 파일" 버튼 클릭
5. 파일 선택 후 현장 등록
6. 트리뷰에서 확인

### 시나리오 2: 파일 변환 실행
1. 트리뷰에서 변환할 파일 확인
2. "▶ 변환 실행" 클릭
3. 진행률 바 확인
4. 로그창에서 진행 상황 확인
5. 완료 후 상태 확인

### 시나리오 3: 채널 설정 변경
1. 트리뷰에서 파일 더블클릭
2. 채널 탭 선택
3. 모드 및 파라미터 설정
4. 저장
5. 다음 변환 시 적용됨

## 💡 UI 특징 및 사용 팁

### 팝업 위치
- 모든 팝업이 메인 창 중앙에 표시됨
- 사용자 편의성 향상

### 색상 코딩
- **파란색**: 주요 작업 (변환 실행)
- **초록색**: 추가/관리 작업
- **회색**: 일반 작업
- **주황색**: 주의 필요 (미등록 파일)

### 상태 표시
- **Ghost 상태**: ⚠️ 아이콘으로 경로 유실 표시
- **배터리**: 파일별 배터리 상태 표시
- **진행률**: 변환 작업 진행 상황

### 키보드 단축키
- **Enter**: 팝업에서 확인
- **더블클릭**: 파일 채널 설정 열기
- **우클릭**: 컨텍스트 메뉴

## 📝 주의사항

1. **미등록 파일**: 로거 등록 시 변환 대상이 아님
2. **Ghost 상태**: 경로가 유실된 항목은 ⚠️ 표시
3. **변환 중**: 버튼이 비활성화되어 중복 실행 방지
4. **삭제 시**: 하위 항목 모두 삭제됨 (확인 필요)
